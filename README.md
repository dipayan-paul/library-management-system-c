# Library Management System in C

A command-line based Library Management System developed in C. The application allows users to manage library records through a simple menu-driven interface while storing data persistently using binary file handling.

## Features

- Add new books
- View all books
- Search books by ID or title
- Update book details
- Delete books
- Persistent storage using `library.dat`
- Menu-driven user interface

## Technologies Used

- C
- Standard C Libraries (`stdio.h`, `stdlib.h`, `string.h`)
- Binary File Handling

## How to Compile

```bash
gcc library.c -o library
```

## How to Run

```bash
./library
```
